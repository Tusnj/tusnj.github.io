# smb 139,445

## Server Connection

### smbclient

``` bash
$ smbclient //ip/tmp

$ smbclient \\\\ip\\ipc$ -U john 

$ smbclient //ip/ipc$ -U john

$ smbclient -L $ip

$ smbclient -N -L //10.10.10.4
```

### smbmap

``` bash
$ smbmap -H $ip

$ smbmap -u <user> -H $ip
```
